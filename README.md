# NARODNI LAUNCHER

NARODNI LAUNCHER is a simple Minecraft launcher for Linux.  
It runs Minecraft version `a1.1.2_01` and allows connecting to servers over the internet.  
Server files are **not stored locally** — all connections happen online.

This version is ready to run as a single file `narodniLauncher.bin`.  
You only need **Java** installed.

## What the launcher can do:

- Runs from one file: `narodniLauncher.bin`
- Works on Linux (Ubuntu, Debian, Fedora)
- Connects to servers online
- Gives housing and resources to new players
- Automatically loads textures and libraries
- Lightweight and fast
- No Python required
- Beginner-friendly

## Project structure

After extraction, your folder should look like this:

- `narodniLauncher.bin` — main launcher file  
- `a1.1.2_01.jar` — Minecraft client  
- `libs/` — library files  
- `libs/nativate/` — LWJGL and OpenAL native libraries  
- `res/` — launcher images and icons  
- `README.md` — this file  

**Do not change folder structure**, the launcher needs these paths to work.
